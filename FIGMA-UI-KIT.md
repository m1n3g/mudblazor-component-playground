# Figma UI Kit

This playground supports the development of the **MudBlazor UI Kit: Design System for Blazor**, a Figma resource for designers and developers building Blazor applications with MudBlazor.

The UI kit closely follows MudBlazor’s component naming, structure, variants, and design principles to create a smoother design-to-development handoff.

[![Open in Figma](https://img.shields.io/badge/Open%20in-Figma-F24E1E?logo=figma&logoColor=white)](https://www.figma.com/community/file/1432408934517112427/mudblazor-ui-kit-design-system-for-blazor)

## ✨ What's Included

- MudBlazor-aligned components
- More than 5,000 component variants
- Auto layout and responsive support
- Variable-based design tokens and theming
- Prototype-ready interactions

##z 🔗 Project Connection

- **Figma UI Kit:** reusable components, variants, themes, design system and interface patterns
- **Component Playground:** interactive properties, states, and live previews
- **Shared purpose:** improve consistency between Figma designs and MudBlazor implementations
